# Weather Diary - Дневник погоды

**Автор:** Мурзагильдин Радмир

## Описание программы

Weather Diary - это GUI-приложение для ведения дневника погоды. Позволяет:
- Добавлять записи о погоде (дата, температура, описание, осадки)
- Фильтровать записи по дате и температуре
- Сохранять и загружать данные в формате JSON
- Валидировать вводимые данные

## Запуск приложения

### Требования
- Python 3.6 или выше
- Установленный tkinter (входит в стандартную поставку Python)
